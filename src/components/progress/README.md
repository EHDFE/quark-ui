## Progress

Progress Component.

### Props
|name|type|default|description|
|---|---|---|---|
|type|string|line|类型，可选 line circle dashboard|
|percent|number|0|百分比|
|showInfo|true|boolean|是否显示进度数值或状态图标|
|strokeWidth|number|10|进度条线的宽度，单位 px|

### Api
